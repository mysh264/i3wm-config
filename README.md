# i3wm-config
My i3wm configurations.

# i3wm config file as a parts
Use this command to merge them.
but first, take a copy from your i3 config file.

```
cp -v ~/.config/i3/config ~/.config/i3/config.bak
```
```
git clone https://github.com/mysh264/i3wm-config.git
```
```
cd i3wm-config/i3wm_parts
```
```
cat * > ~/.config/i3/config
```
